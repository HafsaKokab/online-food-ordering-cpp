# Online Food Ordering System (C++ Project)

This is a simple **C++ project** that implements an online food ordering system.  
The program demonstrates dynamic memory allocation, file handling (both text and binary), and basic CRUD operations for menu items and cart.



## Project Features

- Add, update, remove, and search menu items  
- Sort menu items by price  
- Place orders and manage a shopping cart  
- Cancel or update orders  
- Calculate total amount and checkout  
- Save/load menu and cart to/from **text files** (`.txt`)  
- Save/load menu to/from **binary files** (`.dat`)  



## Files Included

- `main.cpp` : main C++ source code  
- `menu.txt` / `cart.txt` : text files for saving/loading menu and cart data  
- `menu.dat` : binary file for menu items  
- `README.md` : project documentation
